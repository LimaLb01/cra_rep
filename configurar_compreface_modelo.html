Passo 2: Criar Aplicação

  1. Clique em "Create Application"
  2. Nome: facial-recognition
  3. Tipo: Recognition
  4. Clique em "Create"

Passo 3: Obter API Key

  1. Após criar a aplicação, clique em "API Keys"
  2. Copie a API Key gerada

Passo 4: Adicionar Rostos

  1. Clique em "Faces" na aplicação criada
  2. Clique em "Add Face"
  3. Faça upload de uma foto do rosto
  4. Subject ID: Usuário Teste
  5. Clique em "Add"
